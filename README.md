# Gym Progress Track

A fitness progress tracker with web and mobile apps.

## Status

Frontend development in progress. Backend (Supabase) will be added later.

## Features (frontend roadmap)

1. User onboarding
2. Authentication (UI)
3. Dashboard overview
4. Workout logging
5. Exercise database
6. Workout history
7. Bodyweight tracking
8. Progress charts
9. Personal records
10. Profile / settings
11. Data validation
12. Responsive design

## Web app

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
